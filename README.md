# BlogHub

BlogHub is a modern community-driven blogging platform built with Laravel. It enables users to create, publish, and discover engaging content while interacting through likes, comments, and author profiles.
---
## ✨ Features

### 👤 User Authentication
- User Registration & Login
- Secure Authentication System
- Personalized User Dashboard
- Profile Management

### 📝 Blog Management
- Create, Edit, and Delete Blogs
- Publish Rich Blog Content
- Cover Image Support
- Category-Based Organization
- SEO-Friendly Slugs

### 💬 Community Engagement
- Like Blog Posts
- Comment on Blogs
- Interactive Discussions
- Reader Engagement Features

### 📈 Content Discovery
- Browse All Blogs
- Trending Blogs Section
- Featured Authors
- Category-Based Filtering
- Related Blog Recommendations

### 🎨 Modern User Interface
- Responsive Design
- Clean Blog Cards
- Mobile-Friendly Layout
- Modern and Intuitive Experience
